Iconos @expo/ui + Material Symbols

Vectoriales nativos via @expo/ui Icon + @expo/material-symbols. Sin fuentes de iconos — tree-shakeable, solo los iconos que importas.

Instalación
npx expo install @expo/material-symbols
Uso en código
// Importar icono individual
import HomeIcon from '@expo/material-symbols/home.xml';

// Usar con @expo/ui Icon
<Host matchContents>
  <Icon name={HomeIcon} size={24} color={'#fff'} />
</Host>
